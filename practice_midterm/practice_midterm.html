2. Do planes trade ownership?

You might expect that theres an implicit relationship between plane and airline, because each plane is flown by a single airline. Explore this conjecture using data. (Lets assume that the tail number of a plane does not change.)

4b. Average speed

For each flight, compute the average speed of that flight (in miles per hour). Then, for each plane, compute the average of those average speeds. Display it in a histogram. You can use a base R histogram hist or ggplots geom_histogram.

flights_miles_per_hour <- flights_air_time_hours %>% mutate(mph=distance/air_time_hour)
flights_miles_per_hour %>% group_by(tailnum) %>% summarise(avg_mph=mean(mph, na.rm=TRUE)) %>% ggplot(aes(x=avg_mph))  + geom_histogram()
